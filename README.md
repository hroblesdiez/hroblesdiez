## Featured Projects

### [GIS System in Renewable Energy Projects](https://github.com/hroblesdiez/gis_system_in_renewable_energy_projects)
Full deployment of a PostgreSQL/PostGIS spatial database on Windows Server 2019
for renewable energy site selection. Includes custom PostGIS functions to
evaluate and filter land parcels against spatial business rules, multi-user
QGIS editing setup, and automated backup scripts.

### [EGIB Viewer — Polish Cadastral Land Parcel Browser](https://github.com/hroblesdiez/egib-viewer)
Web GIS application that queries live **WFS services** from Polish county
cadastral registers (Ewidencja Gruntów i Budynków). Users can filter parcels
by land use class, registration group, and area across multiple counties,
visualise results on an interactive map, and export filtered data as GeoJSON
(EPSG:2180). Built with vanilla JavaScript, deployed on Netlify with CORS proxy in Oracle Cloud for WFS endpoints.
